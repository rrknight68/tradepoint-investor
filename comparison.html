Why combined Option B return is $14,254,126 not $14,399,703
Option A (single tranche, no sr pref split) returns $14,399,703 WF!B23 combined — more than Option B because there is no fixed-rate pref cost reducing the pool.

Option B is $145,578 less because the sr pref investor takes a fixed $1,560,000 total interest that caps their return and reduces the profit available for the 50/50 split.

The trade-off: the equity tranche IRR jumps from 25.79% to 39.03% because the same above-pref profit pool is split against a smaller equity base ($3,235,062 instead of $7,235,062).
